Git Hub First Web Page Instructions

Sign Up into GitHub

  1. Enter Email
  2. Create Password
  3. Enter a User Name
  4. Choose Product Updates

Create New Repository

  1. Enter Repository Name "username.github.io"
  2. Optional Description
  3. Press Create Repository

Create Index.html and files neede for your web page all in one file

  1. Create index.html file
  2. Creat all other files needed in the same folder
  3. Drag and Drop to GitHub Add files
